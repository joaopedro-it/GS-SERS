# Global Solution - Soluções em Energias Renováveis e Sustentáveis.

## Tema - Central de Monitoramento Espacial 


## Como o projeto funciona

O programa começa mostrando os dados atuais da missão.

Depois, ele verifica se existe algum problema, por exemplo:

- Temperatura acima de 80 graus
- Energia abaixo de 20%
- Comunicação offline
- Painel solar inativo

Quando encontra algum problema, o sistema mostra um alerta.

Em seguida, o programa toma decisões automáticas, como:

- Ativar sistema de resfriamento
- Desligar o módulo de pesquisa para economizar energia
- Tentar reconectar a comunicação com a base

## Relação com sustentabilidade

O projeto considera o uso de energia solar como fonte renovável da missão.
Além disso, quando a energia fica baixa, o sistema economiza recursos desligando módulos secundários.
